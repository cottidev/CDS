# Club des Sciences — Official Website

Official website of the **Club des Sciences** at Lycée Qualifiant Abou Hayane Tawhidi, Bouskoura, Morocco. Built to present the club, its activity poles, team, and make it easy for students to join.

---

## Overview

> *"Let's shape the future together"*

Club des Sciences is a space dedicated to innovation and leadership. This site was built to give the club a modern, professional online presence from its very first day in 2026.

## Features

- **Custom cursor** — dot + animated trailing ring with inertia (desktop only)
- **Particle canvas** — animated network of floating dots in the hero section
- **Hamburger menu** — smooth full-screen overlay menu on mobile
- **Scroll reveal** — sections fade in as they enter the viewport
- **3D tilt effect** — perspective hover effect on activity pole cards
- **Animated counters** — numbers count up when the hero loads
- **Adaptive navbar** — transparent at top, solid on scroll
- **Embedded map** — Google Maps with no API key required (direct coordinates)
- **Fully responsive** — optimized for mobile, tablet, and desktop

---

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript — no frameworks or build tools
- Google fonts
- Google Maps Embed — map without API key
